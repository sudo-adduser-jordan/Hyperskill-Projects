Count the letters

You have a sequence that is made up of 4 letters ('A', 'C', 'G', 'T'). Write a program that counts how many times each letter appeared in the sequence, and prints it. Follow the format in the example below.


Sample Input:
AAGAAGTTCC

Sample Output:
A: 4
C: 2
G: 2
T: 2



Memory limit: 256 MB
Time limit: 15 seconds
